I Dragon

I-dragon is a single page game. In this game a dragon which is controlled by user takes jump to protect itself from the Enemy. As long as player keep the dragon save from Enemy, his score increases in the same manner. For dragon jump player uses the up key of keypad and Right key for moving forward. It is a quite simple game but also that much interesting, if player run this game then he plays it again and again. These are simple games but entertains the mind of user a lot.
